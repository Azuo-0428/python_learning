# 元组(tuple)

- 定义
    - 多个元素按照一定顺序构成的序列
    - 元组是不可变类型，元组类型的变量一旦定义，其中的元素就不能再添加或者删除，而且元素的值也不可修改
    - 变量名 = (x,y,z)
    - 如果一个元组中有两个元素，称之为二元组；（）表示空元组，如果只有一个元素，表示为('hello',)
- 打包和解包操作
    - 打包操作 （多个值赋给一个变量）
        - 变量名 = 值，值，值，····
    - 解包操作 （一个元组赋值给多个变量）
        - 变量，变量，变量 = 元组（变量个数与元组中元素个数须相等）
        - *变量，可以解决变量个数少于元素个数的问题
    - 交换变量的值
      - a,b值互相交换 a,b=b,a
      - a,b,c值互相交换 a,b,c=b,c,a
- 元组的意义
  - 不可变类型更适合多线程环境
  - 不可变类型在创建时间上优于对应的可变类型
